X THREAD — semantic-memory-mcp v0.3.0 launch

1/ Just published semantic-memory-mcp v0.3.0 on crates.io.

An MCP server that gives your AI agent a persistent knowledge base — 33 tools, pure Rust, local-first, no cloud, no API keys, no telemetry.

cargo install semantic-memory-mcp

https://crates.io/crates/semantic-memory-mcp

2/ Every other MCP memory server does: store text, embed it, search it.

semantic-memory-mcp does contradiction detection with belief propagation, adaptive query routing, factor graph reasoning over 4 edge types, community detection, topological gap analysis, bitemporal truth, and memory lifecycle GC.

3/ The contradiction detector is the differentiator.

When your agent stores conflicting facts, the decoder identifies inconsistent items, runs syndrome detection, and computes minimal corrections via belief propagation on conflict graphs.

No other MCP memory server has this.

4/ 33 tools, all local:

- Hybrid BM25 + vector search (RRF)
- Knowledge graph (semantic/temporal/causal/entity edges)
- Factor graph belief propagation
- Community detection (Leiden-inspired)
- Topological gap analysis (Betti numbers)
- Bitemporal truth with supersession tracking
- Memory lifecycle GC with invariant verification
- Blake3-digested receipts for every mutation

5/ No Ollama required.

The default embedder is Candle — a pure-Rust ML framework that runs nomic-embed-text-v1.5 in-process on CPU. Model downloads from HuggingFace on first use, cached after.

SQLite + usearch for storage. Single-file. No external vector DB.

6/ Works with Hermes Agent, Claude Desktop, Cursor, and any MCP-compatible client.

Config:
{
  "mcpServers": {
